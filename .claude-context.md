## Tech Stack

**Framework:**
- Next.js 14.2.35 (App Router)
- React 18.3.1
- TypeScript 5.9.3

**Styling:**
- Tailwind CSS 3.4.19
- Geist Font (Sans + Mono)

**Animation/Interaction:**
- Framer Motion 12.23.26

- GSAP 3.14.2 (ScrollTrigger)

**Development:**
- ESLint
- PostCSS + Autoprefixer

## Design System

**Colors:**

- Foreground: #ffffff (pure white)
- Muted: #86868b (Apple gray)
- Accent: #f97316 (orange - lava-500)
- Surface: #1d1d1f (dark surface)
- Border: #424245 (subtle border)

**Typography:**
- Primary: Geist Sans
- Monospace: Geist Mono
- Features: ss01, ss02, cv01

**Design Philosophy:**
- Apple-level minimalism
- Clean, spacious layouts
- Smooth, purposeful animations
- Focus on content, not decoration

## Important Notes

**Animation System:**
- Background animations (lava/goop) were removed (commit 84c3b87)
- Focus is on content development first
- Animations will be added back later

**Smooth Scrolling:**
- Lenis integrated with GSAP ticker
- Global `window.__lenis` instance available
- ScrollTrigger synced automatically

**Build:**
- Current bundle size: ~192 KB
- Static generation for all pages
- No server-side rendering currently
