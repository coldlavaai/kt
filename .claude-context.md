### Preview URLs
All preview deployments require a bypass secret to view.

**Protection Bypass:**
- Secret stored in: `~/.claude_credentials` as `$VERCEL_BYPASS_SECRET`
- Also available in deployment as: `$VERCEL_AUTOMATION_BYPASS_SECRET`

**How to construct viewable URLs:**
```
{base_url}?x-vercel-protection-bypass=$VERCEL_BYPASS_SECRET
```

**Example:**
```
https://cl-website-jul25-git-redesign-olivers-projects-a3cbd2e0.vercel.app?x-vercel-protection-bypass=$VERCEL_BYPASS_SECRET
```

## Tech Stack

- TypeScript 5.9.3

## Design System

**Colors:**
- Background: #000000 (pure black)
- Foreground: #ffffff (pure white)
- Muted: #86868b (Apple gray)
- Accent: #f97316 (orange - lava-500)
- Surface: #1d1d1f (dark surface)
- Border: #424245 (subtle border)

**Typography:**
- Primary: Montserrat (weights: 300, 400, 500, 600, 700)
- Updated from Geist Sans on 2026-01-05

**Design Philosophy:**
- Apple-level minimalism
- Clean, spacious layouts
- Smooth, purposeful animations
- Focus on content, not decoration
